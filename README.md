## Requirements

You need to install [Node.js](https://nodejs.org)

## How to use it

## SUPABASE_CONFIG


```bash
VITE_SUPABASE_URL=
VITE_SUPABASE_SECRET=
```


```bash
# Install the dependencies
$ npm install
# Run the web server
$ npm run dev
```

The app will be available for access on your browser at (http://localhost:5173)
